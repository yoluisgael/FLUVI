# Carpeta de Logos

Esta carpeta contiene los logos utilizados en FLUVI:

## Archivos necesarios:

- **logo_horizontal.png**: Logo principal que aparece en la pantalla de carga
- **favicon.png** o **favicon.ico**: Icono de la página web (aparece en la pestaña del navegador)

## Recomendaciones:

- **logo_horizontal.png**: Tamaño recomendado 400-600px de ancho, fondo transparente
- **favicon**: 32x32px o 64x64px para mejor compatibilidad
